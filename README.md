

## Installation

1. Clone the repo:
   ```
   git clone https://github.com/SafwanGanz/youtube-dl-cli.git
   cd youtube-dl-cli
   ```
2. In Termux:
   ```
   chmod +x install.sh
   ./install.sh
   ```

## Usage

1. Run:
   ```
   npm start
   ```
2. Enter YouTube URL.
3. Enter `video` or `audio`.
4. Files save to:
   - `/sdcard/Youtube-Dl/Video`
   - `/sdcard/Youtube-Dl/Audio`
